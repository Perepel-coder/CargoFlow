# CargoFlow

1. Запускаем проект CargoFlow.Angular/CargoFlow.Angular.Server
2. Должна выскочить форма авторизации. В базе уже есть (login: perepel, password: 4423) или регистрируем нового.
3. Кликом на строку в списке заказов открывается окно с описанием заказа. Кликом на кнопку "добавить новый заказ" открывается форма создания заказа.
